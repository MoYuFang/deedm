# DEEDM protype


### Installation

`my_deedm` 为项目根文件夹

```
git clone https://github.com/MoYuFang/deedm/ my_deedm
cd my_deedm
my_deedm > npm install
```

### Run dev

```
my_deedm > npm run dev
```

在 `my_deedm/package.json` 中可以看到 `dev` 是别名
```
"scripts": {
"dev": "vite",
"build": "vite build",
"preview": "vite preview"
}
```
